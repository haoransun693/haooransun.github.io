项目背景与愿景

本项目旨在解决传统农业中“问难”与“诊难”的痛点。通过整合 Coze 工作流 与 图像识别 API， 我们构建了一个集“智能问答”与“视觉诊断”于一体的双模态系统。

智能问答 (RAG)

基于专属农业知识库，提供精准的种植、养殖建议，告别模糊搜索。

病害识别 (CV)

上传作物图片，AI 实时识别病害类型，提供防治方案，让科技护航丰收。

核心功能展示

1 Coze 智能问答

技术点：Coze API, RAG (检索增强生成)

亮点：毫秒级响应，支持多轮对话，基于本地知识库的精准回答。

2 图像病害识别

技术点：Canvas, Base64, 图像识别 API

亮点：前端预处理图片，后端精准识别，全流程自动化。

Module 1: Coze 问答核心 

// 1. 数据定义
data: {
  messages: [],
  inputValue: '',
  loading: false
},

// 2. 发起对话请求
fetchAnswer(query) {
  const that = this;
  wx.request({
    url: 'https://api.coze.cn/open_api/v2/chat',
    method: 'POST',
    header: {
      'Authorization': 'Bearer pat_xxxxx',
      'Content-Type': 'application/json'
    },
    data: {
      bot_id: '7536723393757085735',
      user: 'user_' + Date.now(),
      query: query,
      stream: false
    },
    success: function(res) {
      // 3. 智能解析响应
      const answer = that.simpleExtract(res.data);
      that.addMessage(answer, false);
    },
    fail: function() {
      that.addMessage('请求失败', false);
    }
  });
},

// 4. 容错解析函数
simpleExtract(responseData) {
  try {
    if (responseData.data?.messages?.[0]) {
      return responseData.data.messages[0].content;
    }
    return "未获取到有效回答";
  } catch (e) {
    return "解析错误";
  }
}
Module 2: 病害识别核心 

// 1. 选择图片并预处理
chooseImage() {
  wx.chooseMedia({
    count: 1,
    mediaType: ['image'],
    success: (res) => {
      const tempFilePath = res.tempFiles[0].tempFilePath;
      // 显示加载中
      wx.showLoading({ title: '识别中...' });
      // 调用图片转Base64
      this.imageToBase64(tempFilePath);
    }
  });
},

// 2. Canvas 图片转 Base64 (关键预处理)
imageToBase64(imgPath) {
  const that = this;
  const ctx = wx.createCanvasContext('hiddenCanvas');
  
  ctx.drawImage(imgPath, 0, 0, 224, 224); // 统一尺寸
  ctx.draw(false, () => {
    wx.canvasToTempFilePath({
      x: 0, y: 0, width: 224, height: 224,
      destWidth: 224, destHeight: 224,
      canvasId: 'hiddenCanvas',
      success: (res) => {
        // 转换为 Base64 上传
        wx.getFileSystemManager().readFile({
          filePath: res.tempFilePath,
          encoding: 'base64',
          success: (data) => {
            that.uploadToServer(data.data); // 调用上传
          }
        });
      }
    });
  });
},

// 3. 上传至 AI 识别服务
uploadToServer(base64Image) {
  wx.request({
    url: 'https://your-ai-api.com/detect', // 替换为你的识别API
    method: 'POST',
    data: {
      image: base64Image,
      type: 'plant_disease'
    },
    header: {
      'content-type': 'application/json'
    },
    success: (res) => {
      wx.hideLoading();
      // 处理识别结果
      if (res.data.code === 200) {
        wx.navigateTo({
          url: `/pages/result/result?data=${JSON.stringify(res.data)}`
        });
      } else {
        wx.showModal({ title: '错误', content: '未识别出病害' });
      }
    },
    fail: () => {
      wx.hideLoading();
      wx.showModal({ title: '网络异常' });
    }
  });
}
